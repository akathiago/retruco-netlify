Diseño narrativo de un juego de acción (Run & Gun / Beat 'em up / RPG) **que todavía no
está construido**. Es visión a futuro, no documenta lo que hoy existe en el repo.

- `GUION_RETRUCO.md` → guión completo: niveles, personajes, plot.
- `GAMEPLAY_RETRUCO.md` → diálogos, objetivos y mecánicas por nivel.

**Lo que sí está implementado hoy** vive documentado en `../` (ARCHITECTURE, TRUCO_RULES,
SHARED_JS_API, CSS_GUIDE): la landing (`index.html`), el minijuego de truco (`truco/`)
y el character creator (`retrucocharacter/`).
