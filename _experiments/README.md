# _experiments/

Prototipos viejos de animación que **no forman parte del sitio**. Quedaron acá como
referencia/inspiración mientras se buscaba el estilo de los marquees y las transiciones.

- `marquee-*.html`, `variaciones-marquee-*.html` → iteraciones del marquee animado.
  La versión final terminó **inline** dentro de `../index.html`, no se usa ninguno de estos.
- `masking-tape-variaciones.html` → pruebas de efecto "cinta adhesiva".
- `transiciones-test.html` → pruebas de transiciones.

Son standalone y abren solos en el navegador. Algunas referencias a imágenes pueden estar
desactualizadas (apuntan a la estructura vieja de assets). Se pueden borrar sin afectar el sitio.
